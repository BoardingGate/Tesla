Manual de Usuario: BoardingGate Lanzador web para Tesla

Bienvenido a BoardingGate, tu lanzador web personalizado optimizado para la pantalla de tu Tesla.

Vista General de la Interfaz

La interfaz principal se divide en varias áreas:

  • Grid Principal: Muestra tus accesos directos (marcadores) como iconos clicables.
  • Barra Lateral Izquierda: Contiene botones para controlar la visibilidad general, secciones específicas y otras utilidades.
  • Barra Superior: Incluye botones para configuración, personalización, navegación, filtro y acceso a información y versiones.
  • Campo de Filtro: Permite buscar rápidamente en el grid.
  • Identificador de Usuario: Muestra tu ID de usuario si lo has configurado (ej: "North Highland").
  • Banner de Avisos: Muestra mensajes importantes (si están activos).
  • Pie de Página: Información de la versión, enlaces y logo.

Funcionalidad Principal

1. Grid de Marcadores

El área central muestra tus marcadores web.

  • Abrir un enlace: Simplemente haz clic (o tap) en cualquier icono del grid. La URL asociada se abrirá en una nueva pestaña del navegador del coche (generalmente).
  • Espacios Vacíos: Los espacios con un "+" indican celdas vacías donde puedes añadir nuevos marcadores en el modo de personalización.

2. Filtrar el Grid

Usa el campo Filtrar grid... en la parte superior para encontrar marcadores rápidamente.

  • Escribe una o más palabras en el campo. El grid se filtrará mostrando solo los marcadores cuyo nombre contenga todas las palabras escritas (ignorando mayúsculas/minúsculas y acentos).
  • Para limpiar el filtro y mostrar todo de nuevo, borra el texto o pulsa el botón × que aparece junto al campo cuando hay texto.
  • Nota: El filtro interactúa con los botones de ocultación de secciones. Un marcador debe cumplir ambas condiciones (coincidir con el filtro y pertenecer a una sección visible) para mostrarse.

3. Barra Lateral Izquierda

Estos botones controlan la visibilidad y ofrecen utilidades.

  • ON / OFF General: Este botón grande (inicialmente muestra "ON") controla la visibilidad de todos los marcadores del grid. Si lo pulsas (cambia a ), todos los marcadores se ocultarán, independientemente del filtro o de los botones de sección. Púlsalo de nuevo para volver a mostrar los marcadores según su configuración.
  • - PdR,s - iAs - Útil - Varios
    Botones de Sección (-/+ Nombre): Estos botones (PdR,s, iAs, Útil, Varios, etc.) controlan la visibilidad de grupos específicos de marcadores.
    • El signo - indica que la sección está visible. Púlsalo para ocultar los marcadores de esa sección (el signo cambiará a +).
    • El signo + indica que la sección está oculta. Púlsalo para mostrar los marcadores de esa sección (el signo cambiará a -).
    • Puedes asignar qué marcadores pertenecen a cada botón en el modo de personalización (ver más abajo).
  • Tiempo: Abre la web de Ventusky en una nueva pestaña para consultar el tiempo.
  • 4 Recordatorios:
    • El icono de calendario permite gestionar recordatorios.
    • Un número rojo sobre el icono indica la cantidad total de recordatorios pendientes.
    • Un círculo azul con un número en la esquina superior izquierda de la pantalla N indica el número de recordatorios no cíclicos. Haz clic en él para abrir la tabla de recordatorios.
    • Clic corto en el icono del calendario: Abre el modal para crear o modificar un nuevo recordatorio.
    • Pulsación Larga (2 segundos) en el icono del calendario: Abre una tabla con todos tus recordatorios guardados.
  • / Zoom: Cambia la visualización del grid entre 6 columnas (zoom out ) y 4 columnas (zoom in ), haciendo los iconos más grandes o pequeños.
  • / Modo Claro/Oscuro: Cambia la apariencia visual de la página entre un tema claro y uno oscuro . El modo oscuro puede activarse automáticamente por la noche.
  • Scroll Arriba/Abajo: Este botón fijo en la esquina superior izquierda permite ir rápidamente al principio (flecha arriba) o al final (flecha abajo) de la página.

4. Barra Superior

Estos botones ofrecen funciones de configuración, navegación y personalización.

Filtrar grid...
  • Configuración: Abre el modal de configuración, donde puedes gestionar datos de usuario, PIN de acceso, backups y borrar datos locales.
  • Personalizar: Activa los modos de edición y asignación para adaptar el grid a tus necesidades.
  • Inicio/Atrás: Intenta retroceder en el historial del navegador del coche.
  • Recargar: Vuelve a cargar la página completa.
  • Asignar Keyword Filtro: Activa un modo especial para añadir palabras clave a los nombres de los marcadores, facilitando su filtrado posterior.
  • / Clave de Acceso: Si has configurado un PIN en Ajustes, este icono te permite activar () o desactivar () la solicitud del PIN cada vez que cargues la página.
  • Versiones: Muestra el historial de cambios y actualizaciones de la aplicación. El icono será más opaco si no hay novedades desde tu última visita, y brillante si hay actualizaciones recientes.
  • Info/Ayuda: Abre este manual de usuario.

Personalización

BoardingGate te permite personalizar extensamente el grid. Para empezar, pulsa el botón Personalizar .

Se te presentará un menú para elegir el tipo de personalización:

  • Editar URLs / Etiquetas / Reordenar
  • Asignar Botones Ocultación
  • Ver Ayuda (Muestra una guía rápida)

Además, puedes usar el botón de la Llave para el modo de asignación de palabras clave.

Modo 1: Editar URLs / Etiquetas / Reordenar (Borde Amarillo)

Activado al elegir la primera opción del menú. El botón Personalizar tendrá un borde amarillo.

  • Editar Marcador: Haz clic en un icono del grid (o en un espacio vacío "+"). Se abrirá un editor para cambiar Nombre, URL y Favicon. El elemento seleccionado se resaltará en amarillo.
    • Guardar: Guarda los cambios.
    • Resetear: Vuelve a los valores originales del marcador (si existían).
    • Vaciar: Borra el marcador, dejando la celda vacía.
    • Cancelar: Cierra el editor sin guardar cambios.
    • 🔍 / 🖼️: Botones para buscar URL o Favicon en Google.
  • Editar Etiqueta de Sección: Haz clic en uno de los botones laterales de sección (ej: - PdR,s). Se abrirá un editor para cambiar su etiqueta (máx. 7 caracteres).
  • Reordenar Marcadores (Arrastrar y Soltar):
    • PC: Haz clic sobre un marcador y, sin soltar el botón del ratón, arrástralo a la posición deseada (otra celda o un espacio vacío). Suelta el botón.
    • Pantalla Táctil: Mantén pulsado un marcador durante un instante hasta que parezca "flotar" o cambiar ligeramente, luego arrástralo con el dedo a la posición deseada y levanta el dedo.
    • El marcador se insertará en la nueva posición, desplazando los demás. Los cambios se guardan automáticamente al soltar.
  • Salir del modo: Pulsa de nuevo el botón Personalizar (con borde amarillo).

Modo 2: Asignar Marcadores a Botones de Ocultación (Borde Cian)

Activado al elegir la segunda opción del menú. El botón Personalizar tendrá un borde cian.

  • Seleccionar Botón Lateral: Haz clic en el botón de sección (ej: - PdR,s) que quieres configurar. Se resaltará con un borde cian más grueso.
  • Asignar/Desasignar Marcadores:
    • Haz clic en cualquier marcador del grid.
    • Si el marcador no estaba asignado al botón lateral seleccionado, se asignará (se verá normal o resaltado).
    • Si el marcador ya estaba asignado, se desasignará (se verá atenuado).
    • Importante: En este modo, hacer clic en los marcadores NO abre la URL, solo los asigna/desasigna.
  • Cambiar de Botón Lateral: Simplemente haz clic en otro botón lateral para empezar a asignar marcadores a ese grupo.
  • Guardado: Los cambios en las asignaciones se guardan automáticamente al salir de este modo.
  • Visibilidad: Un marcador solo será ocultado por un botón lateral si está asignado a él. Si un marcador no está asignado a *ningún* botón lateral, siempre será visible (a menos que el botón general "ON/OFF" esté en "OFF" o que el filtro lo oculte).
  • Salir del modo: Pulsa de nuevo el botón Personalizar (con borde cian).

Modo 3: Asignar Palabra Clave Filtro (Icono Llave Verde)

Activa este modo pulsando el icono de la llave junto al campo de filtro. El icono tendrá un borde verde.

  • Escribe la Palabra Clave: En el campo Filtrar grid..., escribe la palabra o palabras (se quitarán los espacios automáticamente y se convertirán a mayúsculas) que quieres usar como etiqueta de filtro. Por ejemplo, "trabajo", "viaje", "musica".
  • Asignar al Marcador: Haz clic en el marcador del grid al que quieres añadir esa palabra clave.
  • Resultado: La palabra clave se añadirá al final del nombre del marcador, entre corchetes. Por ejemplo, si añades "MUSICA" al marcador "YouTube", el nombre pasará a ser algo como "YouTube [MUSICA]". Esto te permitirá luego filtrar por "[MUSICA]".
  • Importante: En este modo, hacer clic en los marcadores NO abre la URL, solo les añade la palabra clave escrita en el filtro.
  • Mostrar Todo: Al entrar en este modo, el filtro se limpia temporalmente para que veas todos los marcadores y puedas asignarles palabras clave fácilmente.
  • Guardado: Los cambios se guardan automáticamente cada vez que asignas una palabra clave.
  • Salir del modo: Pulsa de nuevo el icono de la Llave (con borde verde). El filtro que tuvieras antes se reaplicará.

Gestión de Recordatorios

  • Crear/Modificar: Haz clic corto en el icono . Se abre un formulario.
    • Texto: Escribe el texto del recordatorio. Puedes incluir indicaciones de hora/fecha/repetición (ej: "Comprar pan MAÑANA A LAS 10:00", "Revisión CADA 6 MESES", "Llamar Juan CADA VIERNES"). La aplicación intentará interpretar y rellenar los campos de abajo.
    • Voz: Usa el botón del micrófono para dictar el texto.
    • Campos Manuales: Puedes ajustar o introducir manualmente la Hora (HH:MM), Fecha (DD/MM/AAAA), Intervalo (días) y Días de Repetición (Lun, Mar...).
    • Botones: Aceptar (guarda, con cuenta atrás de 25s), Cancelar, Ver Recordatorios (abre la tabla), Ayuda (ejemplos de texto).
  • Ver Todos: Haz una pulsación larga (2s) en el icono . Se abre una tabla con todos los recordatorios.
    • Columnas: Acciones, Texto, Hora, Fecha Próxima, Días Rep., Cada x días, Creado.
    • Acciones por Fila:
      • Visto/Prox: Marca el recordatorio como visto. Si es cíclico (repetición semanal o por intervalo), calcula y guarda la siguiente fecha/hora. Si no es cíclico, lo borra.
      • Posponer: Permite retrasar el recordatorio (15 min, 1 hora, etc.).
      • Anular Ciclo (solo para cíclicos): Borra permanentemente el recordatorio cíclico.
    • Modificar desde la tabla: Haz clic en cualquier parte de la fila de un recordatorio (excepto en los botones de acción) para abrir el modal de modificación de ese recordatorio.
    • Salir: Pulsa el botón SALIR (con cuenta atrás de 60s). La tabla se cierra automáticamente si no hay interacción.
  • Notificaciones: Cuando un recordatorio vence, aparecerá una notificación en pantalla.
    • Modificar: Abre el modal para editar el recordatorio.
    • Posponer: Abre opciones para retrasarlo.
    • Visto / Próximo: Igual que el botón "Visto/Prox" de la tabla.
    • Anular Ciclo (si es cíclico): Borra permanentemente el recordatorio.

Configuración

Pulsa el icono de Configuración para acceder a las opciones avanzadas.

  • Pestaña Usuario: (Opcional) Guarda un ID de usuario (apodo) y datos básicos de tu coche (Modelo, Año, Provincia) en una hoja de cálculo compartida. Puedes permitir o no que el administrador te contacte por MD (Mensaje Directo) a través de tu ID si fuera necesario.
  • Pestaña Clave Acceso: Establece o elimina un PIN local de 4 cifras. Si está activo, se te pedirá al cargar la página. Se guarda solo en tu navegador.
  • Pestaña Backup/Restore:
    • Generar Backup: Crea un código de texto que contiene tus recordatorios, personalización del grid, ajustes de UI y datos de usuario locales. ¡Copia y guarda este código en un lugar seguro!
    • Restaurar Copia: Pega un código de backup previamente guardado aquí. La aplicación intentará restaurar los datos compatibles, sobrescribiendo los actuales.
  • Pestaña Borrar Datos:
    • RESTAURAR EL GRID ORIGINAL: Elimina toda la personalización de marcadores (nombres, URLs, orden) y las asignaciones a botones laterales. Vuelve al grid por defecto. No afecta a recordatorios ni otros ajustes.
    • Borrar TODO: ¡Acción irreversible! Elimina TODOS los datos guardados por BoardingGate en tu navegador (recordatorios, personalización, ajustes, PIN, datos de usuario locales). Usa con precaución y asegúrate de tener un backup si quieres conservar algo.
